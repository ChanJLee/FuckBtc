## How to use

1. Use [DingDing](https://www.dingtalk.com/) to create a robot 

2. clone the repo and 
```shell
cd release
```

3. fetch the token and modify fuck.config
```
ROBOT_TOKEN=your token
```
4. run 
```
./fuck.sh
```
